# Инструкция по работе с галереей и видео для гостиниц

## Добавление фотографий в галерею гостиницы

1. Войдите в админ-панель Django: http://localhost:8000/django-admin/
2. Перейдите в раздел **"Гостиницы"** → выберите нужную гостиницу
3. Внизу страницы редактирования гостиницы вы увидите секцию **"Фото гостиницы"**
4. Нажмите **"Добавить еще Фото гостиницы"**
5. Загрузите фотографию и при необходимости добавьте подпись
6. Установите порядок сортировки (0 - первое фото, 1 - второе и т.д.)
7. Сохраните изменения

## Добавление видео YouTube на страницу гостиницы

1. Войдите в админ-панель Wagtail: http://localhost:8000/admin/
2. Перейдите в **"Pages"** → найдите страницу нужной гостиницы
3. Откройте страницу для редактирования
4. В разделе **"Контент"** нажмите **"+"** чтобы добавить новый блок
5. Выберите блок **"YouTube видео"**
6. Вставьте URL видео YouTube (например: `https://www.youtube.com/watch?v=VIDEO_ID`)
7. При необходимости добавьте подпись к видео
8. Сохраните и опубликуйте страницу

## Поддерживаемые форматы URL YouTube:

- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://www.youtube.com/embed/VIDEO_ID`

## Создание страницы гостиницы в Wagtail

Если у гостиницы еще нет страницы в Wagtail:

1. Войдите в админ-панель Wagtail: http://localhost:8000/admin/
2. Перейдите в **"Pages"** → выберите главную страницу
3. Нажмите **"Add child page"**
4. Выберите **"Страница гостиницы"**
5. Заполните:
   - **Title**: Название гостиницы
   - **Slug**: URL-адрес страницы (например: `grand-hotel-moscow`)
   - **Гостиница**: Выберите гостиницу из списка
6. В разделе **"Контент"** добавьте блоки:
   - **Heading** - заголовки
   - **Paragraph** - текстовые блоки
   - **Image** - изображения
   - **YouTube видео** - видео с YouTube
   - **Embed** - другой встроенный контент
7. Сохраните и опубликуйте страницу

## Просмотр галереи на сайте

После добавления фотографий в админ-панели Django, они автоматически отобразятся на странице гостиницы в Wagtail в виде галереи. 

- Фотографии отображаются в сетке
- При клике на фото открывается модальное окно для просмотра в полном размере
- Видео YouTube встраиваются в страницу и воспроизводятся прямо на сайте
